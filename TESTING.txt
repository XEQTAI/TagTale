TagTale — quick local test
==========================

1) Prerequisites: Node.js, .env with DATABASE_URL + JWT_SECRET + RESEND (or re_dev_placeholder for local).

2) Verify database before anything else:
      npm run db:check

3) Schema + seed:
      npm run bootstrap
   OR:  npx prisma migrate deploy
        npm run db:seed

4) Run app:
      npm run dev
   Open http://localhost:3333/login — email OTP sign-in.

5) Production build (optional):
      npm run build
